Features

Everything you need for reliable, blazing‑fast downloads.

Parallel & Chunked

Splits files into chunks to maximize throughput and resume effectively.

Crash‑safe Progress

Persists progress with SQLite to survive restarts and power loss.

Clean, Minimal UI

Built with Next.js + shadcn/ui for a crisp, distraction‑free experience.

Smart Retries

Automatic backoff and per‑chunk retries for flaky networks.

Proxy & Auth

Works with common auth schemes and proxy setups.

Cross‑platform

macOS, Windows, and Linux builds available in Releases.

Architecture

Ferrix pairs a high‑performance backend with a modern web UI:

  • Rust + Tokio + Tauri backend orchestrates workers, manages concurrency, and ensures safety.
  • Next.js + shadcn/ui frontend provides a reactive, minimal interface.
  • SQLite stores download states and chunk checkpoints for robust resume.
